    UI框架

  • fontawesome - 一套绝佳的字体库和css框架
  • elementUi - 非常优秀的一款web前端UI框架
  • mint-ui - 和Vue绝配的一套H5前端UI框架
  • v-charts.js - 基于 Vue2.0 和 echarts 封装的 v-charts 图表组件，只需要统一提供一种对前后端都友好的数据格式设置简单的配置项，便可轻松生成常见的图表。
  • ant.design - antd 为 Web 应用提供了丰富的基础 UI 组件。
  • vue-blu - Vue-Blu是基于Vuejs和Bulma开发的开源UI组件库。旨在为PC端的前端开发(特别是中后台产品)提供一个快速且灵活的解决方案。
  • echarts - 一个基于 JavaScript 的开源可视化图表库。
  • LULUUI - 一个前端人自己写的UI框架，感觉作者非常的懂设计。
  • Ant-Motion - 基于react的前端动画插件
  • Taro - Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发 微信 / 京东 / 百度 / 支付宝 / 字节跳动 / QQ / 飞书 小程序 / H5 / RN 等应用。
  • vant-ui - Vant 是一个轻量、可定制的移动端组件库，于 2017 年开源。目前 Vant 官方提供了 Vue 2 版本、Vue 3 版本和微信小程序版本。
  • mui - 最接近原生APP体验的高性能前端框架
  • Naive UI - 一个 Vue 3 组件库，比较完整，主题可调。
