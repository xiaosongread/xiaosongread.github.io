简单实现模拟 vue 的数据响应

2023-07-28

主要实现的是修改对象里面的属性，从而使得页面中的显示改变。

采用的是：

  1. Object.defineProperty 这个属性来监听对象属性的改变
  2. window 设置一个变量，来记录当有属性改变的时候，将需要调用的函数记录一下
  3. set 中，当数据属性修改的时候，for 循环来调用相关改变页面的的函数
